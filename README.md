# Secure Login System
A Python-based authentication system that demonstrates secure login using password hashing.
## Features
User login system
Password hashing using SHA-256
Basic authentication validation
## Purpose
This project demonstrates fundamental cybersecurity concepts such as password protection, hashing, and secure authentication systems.
# Tools Used
Python
hashlib (SHA-256)
## Learning Outcome
Understanding how secure login systems work and how passwords are protected using hashing techniques.
